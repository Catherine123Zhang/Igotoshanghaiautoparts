# 展会客户展位标注工具使用说明

## 📋 概述

本工具用于在展会平面图上标注需要拜访的客户位置。每个展厅都有独立的标注页面，支持上传平面图、自动标注、手动调整和导出图片。

## 🚀 快速开始

1. **打开索引页面**
   - 在浏览器中打开 `hall_maps_index.html`
   - 可以看到所有展厅的概览信息

2. **选择展厅**
   - 点击任意展厅卡片，进入该展厅的标注页面

3. **上传平面图**
   - 点击"上传平面图"按钮
   - 选择从展会官网下载或截图的平面图（支持 JPG、PNG 等格式）

4. **自动标注**
   - 点击"自动标注"按钮
   - 系统会自动在平面图上标注所有客户位置
   - 标注会以网格形式分布，您可以根据实际展位位置手动调整

5. **调整位置**
   - 点击并拖拽标注标记，调整到正确的展位位置
   - 可以缩放标记大小（拖拽四角）

6. **导出图片**
   - 调整完成后，点击"导出图片"按钮
   - 保存标注后的平面图（PNG格式，高分辨率）

## 🎨 标注说明

### 颜色编码
- **红色 (#FF4444)**: 大客户（社保人数 ≥ 300人）
- **橙色 (#FF8800)**: 中客户（社保人数 50-299人）
- **黄色 (#FFCC00)**: 小客户（社保人数 < 50人）

### 特殊标记
- **星号 (★)**: 优先客户（产品适合滚镀锌镍合金）
- **粗边框**: 优先客户会有红色粗边框

### 标注信息
每个标注包含：
- 展位号（如 H71）
- 公司名称
- 客户层级和类型

## 💡 使用技巧

1. **获取平面图**
   - 从展会官网下载高清平面图
   - 或使用浏览器截图工具截取互动平面图
   - 建议使用高分辨率图片以获得更好的标注效果

2. **精确定位**
   - 上传平面图后，先找到展位号的位置
   - 手动拖拽标注到对应的展位位置
   - 可以放大浏览器页面（Ctrl/Cmd + 滚轮）进行精细调整

3. **批量操作**
   - 自动标注后，可以一次性调整所有标注位置
   - 建议按照展位号的字母顺序（A-Z）和数字顺序进行排列

4. **保存工作**
   - 调整完成后及时导出图片
   - 建议为每个展厅保存多个版本（如：初始标注版、调整后版）

## 📁 文件说明

- `hall_maps_index.html` - 展厅索引页面（入口）
- `hall_X_X_map.html` - 各展厅的标注页面
- `booth_locations.json` - 展位数据（JSON格式）
- `plating-customers-analysis-report.md` - 完整的客户分析报告

## 🔧 技术说明

- 使用 **Fabric.js** 库进行 Canvas 操作
- 支持拖拽、缩放、旋转等交互操作
- 导出图片支持高分辨率（2x 倍率）
- 所有数据来自客户分析报告
